## 三、Logo 系统

### 3.1 Logo 构成

- 以英文 **Labuki** 为主 Logo，体现全球化定位
- 采用**科技感斜切笔画**，统一 **40° 斜切角度**
- 字母 `a` 和 `i` 使用品牌蓝色点缀，强调 **AI 属性**

### 3.2 四种标准组合形式

| 形式 | 尺寸 | 文件命名 |
|------|------|---------|
| 英文 Logo | 500×500px | `logo_英文-{版本}-500-500.png` |
| 中文 Logo | 500×500px | `logo_中文-{版本}-500-500.png` |
| 横向组合（Labuki 了不起科技） | 800×400px | `logo_组合横-{版本}-800-400px.png` |
| 竖向组合（Labuki / 了不起科技） | 500×500px | `logo_组合竖-{版本}-500-500px.png` |

### 3.3 四种背景适配版本

| 版本 | 文件夹 | 适用场景 |
|------|--------|---------|
| 原色 | `1、原色logo/` | 白色 / 浅色背景 |
| 反白 | `2、白色logo/` | 深色 / 彩色背景 |
| 了不起蓝背景 | `3、了不起蓝背景/` | 品牌蓝底色上使用 |
| 深海蓝背景 | `4、深海蓝背景/` | 深海蓝底色上使用 |

### 3.4 安全空间

- Logo 周围必须保留 **x 单位** 的净空区域（x 为 Logo 标准图中标注的单位）
- 安全空间内**不得放置**任何文字、图形或其他视觉元素
- 确保 Logo 的辨识度和独立性

### 3.5 最小尺寸

| 场景 | 最小尺寸 |
|------|---------|
| 印刷品 | Logo 高度 > **10mm** |
| 数字端 | Logo 高度 > **20px** |

### 3.6 使用禁忌

- 中英文和 Logo 的排列组合是**固定的**，不得自行调整间距或顺序
- Logo 颜色只能使用规定版本，不得自行改色
- 不得拉伸、旋转、添加阴影或特效

## 四、字体系统

### 4.1 字体优先级

| 优先级 | 字体 | 使用场景 | 授权说明 |
|--------|------|---------|---------|
| **首选** | Noto Sans SC | 官网、Web/App UI、品牌正式文件 | 开源免费 |
| **备选** | 钉钉进步体 / 阿里巴巴普惠体 | WPS/Office 无法嵌入 Noto 时 | 开源免费 |
| **办公** | 微软雅黑 | 内部办公文档、制度文件、PPT | 系统自带（仅限桌面显示） |
| **营销扩展** | OPPO Sans | 社交媒体、营销标题、KV 展示 | 免费商用，字间距建议 115%~125% |
| **英文扩展** | Montserrat | 英文场景 | 开源免费 |

### 4.2 使用规则

- 任何代表品牌 Logo、公司名称的正式文件、合同抬头，**必须使用首选字体**
- 内部协作人员无法安装品牌字体时，提供备选字体方案
- 扩展字体需提交《扩展字体使用申请》，说明项目和使用位置（现阶段可口头确认）

### 4.3 版权红线

| 行为 | 状态 |
|------|------|
| 将苹方安装到 Windows 上使用 | **严禁** |
| 苹方/微软雅黑用于商业印刷品、Banner | **严禁** |
| 在非苹果平台嵌入苹方字体文件 | **严禁** |
| 苹方/微软雅黑作为品牌 Logo 或 UI 图标字体 | **严禁** |
| 不确定某字体能否商用时 | 优先选择上表中的开源字体 |

## 五、视觉风格 & UX/UI 原则

### 5.1 三大设计原则

| 原则 | 关键词 | 在 PPT 中的体现 |
|------|--------|----------------|
| **信赖感（Trust）** | 稳重、专业、可靠 | 深色封面/结尾页，稳重配色 |
| **高效（Efficiency）** | 清晰、直观、快速获取信息 | 明确的信息层级，一目了然的布局 |
| **克制（Minimalist）** | 留白、简洁、不堆砌 | 大量留白，装饰元素少而精 |

### 5.2 视觉语法（Core Syntax）

手册提到的两个核心概念：

- **核心语法（Core Syntax）**：基于几何图形（斜切角、圆、线条）的品牌视觉语言
- **生成式设计系统（Generative Design System）**：AI 驱动的可扩展视觉方案

在 PPT 设计中的具体体现：

| 元素 | 实现方式 |
|------|---------|
| 斜切几何装饰 | 封面页的倾斜矩形色块（呼应 Logo 40° 斜切） |
| 圆形元素 | 章节分隔页、封面页的装饰圆 |
| 三点装饰 | 品牌识别性图案，用于封面/章节页/结尾页 |
| 蓝色竖条 | 标题左侧的品牌蓝竖条标记 |
| 卡片容器 | 干净的白色/浅灰卡片 + 微妙阴影 |
| 细线分隔 | 浅灰色分割线用于区隔内容 |

## 六、PPT 专用设计规范

### 6.1 页面类型与配色

| 页面类型 | 背景色 | Logo 版本 | 文字基色 |
|---------|--------|----------|---------|
| 封面页 | `#020617`（了不起黑） | 反白横向组合版 | 白色 |
| 目录页 | 左栏 `#4BB1FF` + 右栏白色 | 反白横向组合版（蓝色区） | 白色 + 黑色 |
| 章节分隔页 | `#4BB1FF`（了不起蓝） | 无 | 白色 |
| 内容页（白底） | `#FFFFFF` | 原色横向组合版 | 了不起黑 |
| 内容页（浅灰底） | `#F8FAFC` | 原色横向组合版 | 了不起黑 |
| 结束页 | `#020617`（了不起黑） | 反白横向组合版 | 白色 |

### 6.2 文字层级

| 层级 | 字号 | 字重 | 颜色 |
|------|------|------|------|
| 封面标题 | 42pt | Bold | `#FFFFFF` |
| 章节标题 | 38pt | Bold | `#FFFFFF` |
| 页面标题 | 22pt | Bold | `#020617` |
| 副标题 / 说明 | 11pt | Regular | `#475569` |
| 正文标题 | 16pt | Bold | `#020617` |
| 正文内容 | 12~13pt | Regular | `#475569` |
| 页脚 | 7~8pt | Regular | `#94A3B8` |

### 6.3 间距规范

| 区域 | 最小间距 |
|------|---------|
| 幻灯片边缘 | 0.5 英寸（约 1.27cm） |
| 内容模块之间 | 0.3~0.5 英寸 |
| 卡片内部边距 | 0.2~0.3 英寸 |
| 页脚分割线上方 | 至少 0.3 英寸 |

### 6.4 装饰元素使用指引

| 元素 | 使用规则 |
|------|---------|
| 几何装饰（斜切矩形、圆） | 仅用于封面、章节分隔、结尾页，内容页保持干净 |
| 三点装饰 | 最多 3 个，渐变透明度，用于过渡区域 |
| 蓝色竖条 | 固定宽度 0.06 英寸，高度 0.5 英寸，位于标题左侧 |
| 卡片阴影 | 微妙外阴影（blur 10, offset 3, opacity 8%） |
| 顶部强调线 | 卡片顶部 0.05 英寸的品牌蓝/深海蓝色条 |
